# Python Scripts Portfolio

Welcome to my Python portfolio. I'm a self-taught programmer from Venezuela, 61 days of consistent practice. Here you'll find my most useful scripts.

## Projects

### 1. Password Generator (`generator.py`)
- Generates secure passwords using two user words + random symbol + random number
- Saves passwords encoded in Base64
- View and delete saved passwords
- Passwords persist between sessions

### 2. Task Manager (`tareas.py`)
- Simple To-Do List manager
- Add, view, complete, and delete tasks
- Tasks are saved automatically

### 3. Guessing Game (`guess.py`)
- Guess a random number between 1 and 100
- 5 attempts per game
- Hints: "too high" or "too low"
- Shows the secret number if you lose

### 4. Countdown Tracker (`countdown.py`)
- Track days remaining for important events
- Add, view, and delete events
- Shows: "X days remaining", "TODAY", or "X days ago"
- Events are saved permanently
