# Python Scripts Portfolio

## 📁 Scripts

### 1. generator.py
**Password generator** with Base64 encoding.  
- Asks for two words
- Adds random symbol and number
- Saves passwords encoded
- View and delete saved passwords

### 2. tareas.py
**To-Do List manager** with file persistence.  
- Add, view, complete, delete tasks
- Saves to `tareas.txt`

### 3. quess.py
**Number guessing game**.  
- Random number between 1-100
- 5 attempts with hints
- Shows secret number if you lose

## 🛠️ How to use
```bash
python generator.py
python tareas.py
python quess.py
